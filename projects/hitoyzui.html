> HI-Toyz

Demo

The HI-Toyz UI Design project focuses on creating the user interface for an e-commerce platform that sells various types of toys. This design was crafted using Figma and served as the blueprint for developing the front-end of the HI-Toyz website. As the first step in the development process, this project allowed me to practice designing user-friendly interfaces, organizing components, and visualizing the final product before transitioning to code. It was an essential phase in ensuring the website's structure and aesthetics align with user expectations.
