Repository

Car-merce is a mobile application built using Java in Android Studio, designed to provide a simple platform for browsing and selecting cars. In this application, users can view a list of available cars, add their chosen cars to a cart, and manage their selections. The application integrates with a database to store and retrieve product information, and users can view their cart for a seamless shopping experience. Although Car-merce is still under development, it has been a valuable project for practicing mobile application development and working with databases.
