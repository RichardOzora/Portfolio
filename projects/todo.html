> To-Do List

Demo

The To-Do List project is a simple web application that helps users manage their tasks effectively. Users can add tasks, mark them as completed, and delete them when no longer needed. This project was built using React and served as my first hands-on experience with the framework. Through this project, I learned the basics of React, including components, state management, and handling user interactions, making it an essential step in my journey to mastering modern front-end development.
