> HI-Toyz

Demo

HI-Toyz is an e-commerce website designed to showcase and sell a variety of toys. The project began with creating a user-friendly UI design in Figma, which was then brought to life using HTML, CSS, and JavaScript. While the website currently lacks a backend, it serves as a practical exercise to hone my front-end development skills and explore essential web technologies. This project reflects my effort to merge design concepts with functional implementation, making it a valuable learning experience in building responsive and visually appealing websites.
